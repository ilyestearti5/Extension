# Privacy Policy for CleanWeb Customizer

**Last Updated: December 4, 2025**

## Overview

CleanWeb Customizer is committed to protecting your privacy. This extension does not collect, store, or transmit any personal data to external servers.

## Data Storage

- All data (hidden element rules, preferences) is stored locally in your browser using Chrome's storage API
- Data is synced across your devices only through Chrome's built-in sync feature (if you're signed into Chrome)
- No data is sent to third-party servers or developers

## Permissions Used

### Storage Permission

Used to save your hidden element rules and preferences locally in your browser.

### Active Tab Permission

Used to access the current webpage to hide elements you select.

### Scripting Permission

Used to inject content scripts that apply your hiding rules to web pages.

### Context Menus Permission

Used to add right-click menu options for hiding elements.

### Host Permissions (<all_urls>)

Required to allow the extension to work on any website you visit. The extension only modifies pages when you explicitly use it.

## Data Collection

CleanWeb Customizer does NOT:

- Collect any personal information
- Track your browsing history
- Send data to external servers
- Use cookies or analytics
- Share data with third parties

## Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the extension's store listing.

## Contact

For questions about this privacy policy, please contact through the Chrome Web Store support page.
