# CleanWeb Customizer - Keyboard Shortcuts

## Quick Reference

| Shortcut       | Action           | Description                               |
| -------------- | ---------------- | ----------------------------------------- |
| `Ctrl+Shift+H` | Toggle Selection | Start/stop element selection mode         |
| `Ctrl+Shift+U` | Undo Last        | Restore the last hidden element           |
| `Ctrl+Shift+T` | Toggle Rules     | Enable/disable all rules for current site |
| `ESC`          | Cancel           | Exit selection mode without hiding        |

_On Mac, use `Cmd` instead of `Ctrl`_

## How to Use

### Hiding Elements

1. Press `Ctrl+Shift+H` to start selection mode
2. Hover over elements to see info (tag, size, match count)
3. Click on the element you want to hide
4. Element is hidden immediately and on future visits

### Managing Hidden Elements

- Press `Ctrl+Shift+U` to quickly undo the last hidden element
- Press `Ctrl+Shift+T` to temporarily disable/enable all rules
- Use the popup or right-click menu for more options

## Context Menu Options

Right-click anywhere on a page:

- **Hide this element** - Start selection mode
- **Hide this element permanently** - Start selection mode (same as above)
- **Undo last hidden element** - Restore last hidden element
- **Show all hidden elements** - Temporarily show all hidden elements (5 seconds)

## Tips

- **Preview before hiding**: Hover to see how many elements will be affected
- **Undo mistakes**: Use `Ctrl+Shift+U` immediately if you hide the wrong element
- **Temporary disable**: Use `Ctrl+Shift+T` to test if rules are causing issues
- **Badge indicator**: Look at the extension icon to see how many rules are active

## Customizing Shortcuts

1. Go to `chrome://extensions/shortcuts`
2. Find "CleanWeb Customizer"
3. Click the pencil icon to change shortcuts
4. Choose your preferred key combinations

## Notes

- Shortcuts work on all pages except Chrome internal pages (chrome://, chrome-extension://)
- Some websites may override keyboard shortcuts
- Press `ESC` at any time to cancel selection mode
